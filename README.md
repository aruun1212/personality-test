# 🤓☝️ 留步，你有话说

> 基于大五人格 × 暗黑三角的双面人格测试

一个 Z 世代风格的性格测试网站，帮你找到你的「表面 OS」和「暗面 OS」。

## ✨ 特点

- 📝 36 道情境选择题（非无聊的李克特量表）
- 🧠 基于大五人格 (OCEAN) + 暗黑三角模型
- 🎭 32 种人格类型 × 双面 OS 揭示
- 📊 五维雷达图 + 暗黑指数可视化
- 📸 一键生成分享卡片（含二维码）
- 📱 移动端优先，暗黑霓虹主题
- 🚀 零依赖单文件部署

## 🔗 在线体验

👉 [https://aruun1212.github.io/personality-test/](https://aruun1212.github.io/personality-test/)

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（单文件）
- html2canvas（截图生成）
- qrcode-generator（二维码生成）
